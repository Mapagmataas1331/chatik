// Этот файл содержит компонент MainActivity, который является главной активностью приложения.

// MessageService.kt
// Этот файл содержит интерфейс MessageService, который описывает методы для работы с сообщениями чата.

// UserService.kt
// Этот файл содержит интерфейс UserService, который описывает методы для работы с пользователями приложения.
